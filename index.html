a hailstone sequence is a sequence of numbers generated by the following process:

if n is even, divide n by 2.

if n is odd, multiply n by 3 and add 1

the collatz conjecture says that no matter which number you start with, you will eventually reach 1.

what this sonification / visualization does is treat each number in a hailstone sequence as a frequency.

though every sequences sounds slightly different, each hailstone sequence possesses the same almost-harmonious, slightly eery sound. this is because dividing the frequency of a note in half lowers that note by an octave, and multiplying a note by 3 raises that note by about an octave and a fifth. the additional +1 hz (as well as the fact that in a twelve-tone equal temperament system, an octave and a fifth is not exactly 3 times the frequency) leads to subtle shifts over time, most noticeable when the release time is high.
